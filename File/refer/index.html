수술 후 처치

일정 기간이 지나면 어떤 처치가 이루어지는가

POD ( 수술 후 일 수)

  • POD 4
    : 내복약 조제
  • POD 10 ~ 14
    : 봉합사 제거
  • POD 21 (3주)
    : Xray 1차
  • POD 42 (3주)
    : Xray 2차

라인 잡은지 3일~4일차 라인 교체

한축 바이탈 체크 목록

  • Resp 호흡수

    (respiration rate)
    편안한 상태에서 측정하는 것이 좋음 (수면상태)
    만지지 않고 재는 것이 좋다
    1분간 측정해야하지만 시간상
    15초 X 4
    20초 X 3

  • Pulse 심박

    1분간 측정해야하지만 시간상
    15초 X 4
    20초 X 3

  • BT 체온

    (Body Temperature)

  • NIBP 혈압

    (systol) 이완기
    (mean) 평균
    (Diastol) 수축기 병원에 있는 기계

  • MMC 잇몸 색깔

    blue 파란색 (위험)
    Pink 핑크색( 정상)
    Pale pink 약간 핑크색
    red 빨간색 ( 흥분을 했을때 잇몸질환일 경우)

  • CRT 모세혈관 재충만 시간

    <1sec : 정상
    >1sec : 비정상(심각 탈수)

  • defecation 대변

    데페케션
    정상 ( normal 로 체크 )
    diarrhea 설사

  • Urination 소변 정상

    정상 ( normal 로 체크 )
    hematuria 혈뇨

  • Vomiting 구토
  • IV line

    몇일에 잡았는지 유지는 잘 되고 있는지 발이 부었는지

  • BW 체중

    체중 모니터링 이 중요한 아이일 경우
    ( 신장병 - 뇨 배출을 잘 해야 할 경우)
    ( 복막에 복수가 차 배출했던 아이들일 경우)

  • Skin Torgur 탈수 체크

    피부를 꼬집었을때 피부회복속도를 보는 것
    이것만으로 정확하게 측정할수 없음

수술명(진단명) 용어

  • MPL 슬개골탈구 *진단명
  • FHO 고관절탈구 *수술명
  • Arhtrossopy 관절경 *진단용
    Arhtro / As (관절 내시경 확인용)
    L 왼쪽(좌측) / R 오른쪽(우측) / Bi 양측
  • TPLO 십자인대 수술
  • CCLR 십자인대 파열 *진단명
  • hemilominetomy 디스크수술 (IVDD)-*진단명
  • AAI 환축추 아탈구

    두 개골과 만나는 첫번째 척추와 두번째 척추가 외부적인 충격으로 인해 첫번째 척추를 지나가는 신경을 두번쨰 척추가 누르는 경우

    *아탈구와 탈구를 다름
    덜렁덜렁 걸려있는 경우 "아탈구"
    완전이 빠져있는 경우 "탈구"
  • FMD 두개골성형술 (키아리(경련) 일때 ) * 수술방법

    척추(Spinal cord) 머리가 눌려 나가야할 뇌수(?) 뇌압에 올라가 발작하는 증상을 보안해 줄수 있는 수술

  • Spinal cord 척수
  • Tumor 종양
  • Cervical 경추
  • Thoraclumda 흉추
  • Lumbar 요추
